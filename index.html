A Novel Approach of CNN Patterns Extraction for Bangla Handwriting to Bangla Braille Notation

Braille is the only tactile dotted method to learn for visually impaired people. They can read what is written there by sliding their finger over the dots. They can't read handwritten documents or books that haven't been converted into braille. Each braille cell represents a different alphabet or symbol. In the braille method higher education resources are inadequate which are essential for their existence. Certified professionals are necessary to translate regular language into the braille notation. It’s a very costly and time-consuming process to overcome this deficiency here proposed automatic braille conversation approach. It is capable of transforming handwritten sheets into braille notation. This entire document goes through the braille system and its grammatical rules in great depth. This research introduces an object detection model Faster-RCNN to propose boundary boxes over Bangla characters and use ten different CNN models to classify those characters and also introduce Bengali braille conversation approach. After creating braille notations that can be printed in a legible manner for persons who are blind or visually challenged.

Automatic Number Plate Detection and Authentication

Object Detection YOLO v4 & YOLO v5 - Object detection is a very sophisticated technology to work on nowadays. In this project, we specifically focused on solving the problem of License plate detection and license plate class identification for authentication. For both, we used the YOLO algorithm to solve these problems. For the License plate detection model here we used YOLO v4 and for the Number plate class identification model used YOLO v5. The Dataset is used in License plate detection is google’s Open Image Dataset to get YOLO formatted Labeled images. And for Number Plate Authentication model, we have created a dataset by labeling the images.
